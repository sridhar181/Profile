There are many algorithms out there which do not use Deep Learning methodology for Image Segmentation. The traditional methods include comparing pixels among each other. Those traditional methods cannot be customized for a specific purpose. Threshold based segmentation, Edge based segmentation , Clustering based segmentation are some of the methods that are already existing which do not use Neural Networks. The proposed system uses Neural networks for Image Segmentation. These methods can be customized for a specific purposes. R-CNN, Fast R-CNN and Faster R-CNN are compared and Faster R-CNN is picked for the implementation. Faster R-CNN works as unified model for image segmentation. This Faster R-CNN architecture consists of the RPN as a region proposal algorithm and the Fast R-CNN as a detector network.
